# Nuza

## Getting Started with the Frontend

To start the frontend of the Nuza application, follow these steps:

1. Navigate to the frontend directory:
   ```bash
   cd Frontend
   ```

2. Install the necessary dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

Your frontend application should now be running and accessible at `http://localhost:3000` (or the port specified in your Vite configuration).
